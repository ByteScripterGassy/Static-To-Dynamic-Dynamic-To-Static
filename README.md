# Static-To-Dynamic-Dynamic-To-Static
Good for when you hate doing all the work just to convert your Static Address To Dynamic and Dynamic Address To Static

This is a tool that will automatically convert your Static Address To a Dynamic Adress, same goes in reverse. This is a tool meant for Reverse 
Engineering but can just be used to transfer offsets to different bases.

## Example One:

```
  Static Base: Static Base From Static Disassembler
  Dynamic Base: Dynamic Base From Debugger

  (The above only appears once the rest gets looped so you dont have to bother putting it in again)

  Static [S] or Dynamic [D]: S
  Static Address: Static Address From Static Disassembler
  Dynamic Address: A Dynamic Address For Your Debugger

  (Waits 4 seconds before clearing and continuing the loop)
```

## Example Two:

```
  Static Base: Static Base From Static Disassembler
  Dynamic Base: Dynamic Base From Debugger

  (The above only appears once the rest gets looped so you dont have to bother putting it in again)

  Static [S] or Dynamic [D]: D
  Dynamic Address: Dynamic Address From Debugger
  Static Address: A Static Address For Your Static Disassembler

  (Waits 4 seconds before clearing and continuing the loop)
```
